HTML Basic

HTML Links

HTML linker er definert som a-tag.

This is a link

Linken sin destiansjon er spesifisert i href atributten.
Atributter blir brukt til tileggsinformasjon om HTML-dokumenter.

HTML Bilde

HTML-bilder er definert med img-tagen.
Kildefila (src), alternativ tekst (alt) og størrelse(width og height) er gitt som atributter.

HTML Elementer

Et HTML element består vanligvis av en start-tag og en stopp-tag, med innhold i midten.
HTML-elementet er alt fra start-tagen til slutt-tagen.
Det er ikke alle HTML-elementene som har innhold, og har heller ikke noen slutt-tag, som break-tagen.

Nested HTML Elementer

HTML-elementer kan bli "nested" (elementer kan inneholde elemeneter).
Alle HTML-dokumenter består av "nested" HTML-elementer. Dette eksempelet innholder fire HTML-elementer.
